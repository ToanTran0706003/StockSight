# StockSight Demo Script

1. Open the watchlist and show live quote cards.
2. Click `AAPL` to show candlesticks, indicator overlays, RSI/MACD panes, AI signal, and news sentiment.
3. Open Backtest, run the MACD strategy, and explain return, Sharpe, drawdown, and trade log.
4. Register or login as the demo user.
5. Open Portfolio, create a virtual portfolio, buy AAPL, and show P&L.
6. Open Alerts, create an AAPL threshold alert, and explain SignalR notification flow.
7. Open GitHub Actions and show the green CI workflow.

Record this flow with OBS or Loom and add the generated GIF/video link to the README when a hosted demo URL is available.
